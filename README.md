# BGraphic

A lightweight experimental renderer written in **C++** using **Direct3D 12**, designed to render HTML‑like 2D content.

This Project is intended to used in a D3D12 Project, and is not a standalone program. 

Template syntax similar to Angular.

Two dependancies:
- [stbi_image.h](https://github.com/nothings/stb/blob/master/stb_image.h)
- [CPPLib Repo](https://github.com/BrandonZ3/CPPLib).

This project is still a work in progress.

---

## Requirements
- Visual Studio (MSVC toolchain)
- DirectX 12 runtime
